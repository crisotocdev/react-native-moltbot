FASE 1 — CONECTAR LOGIN CON SERVIDOR (PRIORIDAD)

Objetivo: La app valida token contra el servidor Rust.

Crear endpoint en servidor:

POST /auth/verify

Body:
{ "token": "ABC123" }

Respuesta OK:
{ "ok": true, "role": "admin" }

Respuesta error:
{ "ok": false }

Probar endpoint en navegador o curl:

http://127.0.0.1:8080/auth/verify

Crear archivo en la app:

/services/api.ts

Funciones:

verifyToken(token)

setApiBase(url)

Modificar login.tsx

Al presionar Guardar:

Llamar verifyToken

Si ok → guardar token

Si error → mostrar alerta

Guardar datos con AsyncStorage:

moltbot_token

moltbot_api_base

Redirigir a Home:

router.replace("/home")
